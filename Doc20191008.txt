Клонирование репозитория на локальный диск:
git clone адрес

Показать разницу между облаком и локальной версией:
git status

Добавить новые файлы к отслеживаемым/включить в список передаваемых в репозиторий:
git add файл
git add -A
git add --all

Подготовка к передаче в облако(репозиторий) наших изменений:
git commit -m "сообщение описывающее что синхронизируется с облаком, какие изменения, состояние разработки"

Отправка изменений в облако:
git push

Синхронизация с облаком/получение свежей версии из облака на локальный диск:
git pull
